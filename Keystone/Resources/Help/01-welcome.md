# Welcome to Keystone

Keystone is a local-first, Apple-native life management and knowledge system. It's a structured object database with a rich page editor on top — somewhere between Notion and the Mac Finder.

You don't keep notes in Keystone. You keep **records**: a person, a pet, a vehicle, a home, a document, an event. Each record has typed properties, a page body, files, tags, and links to other records.

## What makes it different

- **Local-first.** Your data lives in a single SQLite file on your device. No login required. CloudKit sync is opt-in.
- **Structured.** Everything is a typed record in a typed database. Filtering, linking, and sorting work because the data is real.
- **Apple-native.** SwiftUI throughout. Runs on Mac, iPhone, and iPad with the same code and the same data.
- **Exportable.** Files are real files on disk. The database is plain SQLite. There's no lock-in.
