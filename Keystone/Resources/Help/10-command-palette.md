# Command palette

`⌘K` (Mac / iPad hardware keyboard) or the **Search** tab (iPhone) opens the command palette — a fast, keyboard-driven search across your workspace.

## What's indexed

The palette searches:

- **Databases** by name (e.g., typing `peo` finds the People database).
- **Records** by title (e.g., typing `ele` finds Eleanor Marsh).
- **Quick capture actions** (`New person`, `New event`).

## Keyboard

- **Type** to filter.
- **`↑` / `↓`** to move selection.
- **Return** to open the selected result.
- **Esc** to close.

## Picking a result

- Pick a **database** → navigates to that database's main view.
- Pick a **record** → opens the record's detail view.
- Pick an **action** (e.g., `New person`) → opens Quick capture pre-set to that kind.

## What's NOT indexed (yet)

- Property values (e.g., searching for a phone number doesn't surface the matching person).
- Block content (page-body text isn't searched).
- Tags by name.
- File contents.

These will come as full-text-search lands.
